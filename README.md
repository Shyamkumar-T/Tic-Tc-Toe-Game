# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**.  
This project focuses on front-end development basics like DOM manipulation, event handling, and game logic.

## Features
- Two-player game
- Responsive design
- Win detection
- Draw detection
- Restart game option
- Clean and simple UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Play
1. Open the game in a browser.
2. Players take turns clicking on the boxes.
3. The first player to get 3 in a row wins.
4. If all boxes are filled and no one wins, the game ends in a draw.
5. Use the restart button to play again.
